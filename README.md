### ğŸŒ± Floralink
For my bachelor thesis in Landscape Use and Nature Conservation, I tried to build a showcase for using existing data of plant occurences and plant taxon information to analyze ecological factors. In the relatively small world of flora and vegetation experts and enthusiasts there exist only a few old or complicated tools for working with plant occurrence and vegetation plot data. In order to show what can be done with today's databases I developed a prototype web app called [Floralink](https://github.com/floralink). Here you can analyze plant occurrence data fetched from an existing database in combination with taxon specific properties like Ellenberg indicator values (describing ecological site factors). It's supposed to act as a showcase and is not finished by any means. Feel free to try out the [online demo ğŸ¡­](https://flora-link.de).
